# IoTech Smart Products and Installation Services
Technopreneurship Project from PUP Sta. Mesa


## Installation
1. Clone the repository
```bash
git clone
```

2. Install the dependencies
```bash
npm install
```

3. Run the app
```bash
npm start
```

## Usage
1. Open the app
2. Click the "Add Product" button
3. Fill up the form
4. Click the "Submit" button
